# TypeScript Step Learning

结构：
- `steps/`：原理讲解
- `practice/`：对应练习（`.ts`）
- `progress.md`：学习打勾
- `tsconfig.json`：本学习包编译配置

建议执行：
1. `npx tsc -p ts-step-learning --noEmit`
2. 逐步查看 `practice/*.ts`

学习顺序：
1. 类型系统观念
2. 联合类型与收窄
3. interface vs type
4. 泛型
5. keyof 与映射类型
6. 条件类型与 infer
7. unknown/any/never
8. 声明文件
9. tsconfig 严格模式
10. 渐进迁移
