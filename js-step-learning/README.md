# JavaScript 学习包（按步骤）

这套内容分成两段：
- 基础巩固（Step 01-10）
- 核心进阶（Step 11-22）

每一步都包含：
- 一个概念讲解文件（`steps/`）
- 一个可运行练习文件（`practice/`）

建议节奏：
1. 先看 `steps/xx-*.md`
2. 完成对应 `practice/xx-*.js`
3. 执行 `node practice/xx-*.js`
4. 在 `progress.md` 勾选完成项

学习顺序：
1. 执行上下文
2. 作用域链
3. 闭包
4. this 绑定规则
5. 原型链
6. 修改原型返回结果
7. Symbol.toStringTag
8. 获取纯净 toString
9. iframe 内 window 是否相同
10. 判断普通对象函数
11. 数据类型与隐式转换
12. let/const 与 TDZ
13. 深浅拷贝
14. 属性描述符与对象冻结
15. 事件循环（宏任务/微任务）
16. Promise 链与错误冒泡
17. async/await 与并发控制
18. 迭代器与生成器
19. Proxy 与 Reflect
20. Map/Set/WeakMap
21. 错误处理与自定义错误
22. 模块化（CommonJS 与 ESM）
