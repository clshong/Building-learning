# Step 06 修改原型的返回结果

目标：会通过修改原型方法改变实例调用结果。

## 关键点
- 实例调用原型方法时，读取的是“当前原型上的方法实现”。
- 修改 `Ctor.prototype.method` 后，已有实例通常也会受影响。

## 示例
```js
function User(name) {
  this.name = name;
}

User.prototype.label = function () {
  return `User:${this.name}`;
};

const u = new User("A");
console.log(u.label());

User.prototype.label = function () {
  return `Member:${this.name}`;
};

console.log(u.label());
```

## 通关标准
- 你能解释为什么旧实例也会输出新结果。
