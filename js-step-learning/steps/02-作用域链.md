# Step 02 作用域链

目标：掌握变量查找路径。

## 关键点
- 变量查找从当前作用域开始，逐层向外。
- 查找规则由“定义位置”决定（词法作用域）。

## 观察代码
```js
const name = "global";

function outer() {
  const name = "outer";

  function inner() {
    console.log(name); // outer
  }

  inner();
}

outer();
```

## 练习
1. 把 `inner` 返回出去再在外部调用，观察是否还是 `outer`。
2. 解释为什么不是 `global`。

## 通关标准
- 你能用“由定义位置决定，不由调用位置决定”解释输出。
