# Step 03 闭包

目标：知道闭包的本质和常见用途。

## 关键点
- 闭包 = 函数 + 定义时捕获的词法环境。
- 外层函数结束后，被引用的变量仍然可访问。

## 观察代码
```js
function createCounter() {
  let count = 0;
  return function () {
    count += 1;
    return count;
  };
}

const next = createCounter();
console.log(next()); // 1
console.log(next()); // 2
```

## 练习
1. 实现 `createCounter(start)`，初始值可配置。
2. 尝试创建两个 counter，确认互不影响。

## 通关标准
- 你能解释“为什么 count 不会被重置”。
